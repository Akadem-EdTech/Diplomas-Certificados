🔑 Cómo crear un Token de Acceso Personal (PAT)

    

      1. Ir a github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)

      2. Clic en "Generate new token (classic)"

      3. Nombre: AKADEM Diploma System

      4. Expiration: No expiration (o 1 año)

      5. Scopes requeridos: ✅ repo (acceso completo al repositorio)

      6. Clic en "Generate token" y copiar el valor

    

    
⚠️ El token se guarda encriptado en localStorage. No compartir el archivo HTML con el token incluido.
